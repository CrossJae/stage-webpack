### webpack脚手架

```
.
├── app 项目目录
│   ├── assets 多媒体资源
│   ├── components 组件
│   ├── entry.js 入口文件
│   ├── lib js库
│   └── template html模版
├── build 打包
│   ├── common.js
│   ├── images
│   ├── index.html
│   ├── main.css
│   └── main.js
├── package-lock.json
├── package.json
├── webpack.config.dev.js 开发配置
└── webpack.config.pro.js 生产配置
```
